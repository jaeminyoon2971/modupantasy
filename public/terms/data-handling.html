유저 데이터 처리 약관

5.1 작품 데이터 소유권

사용자가 작성하고 저장한 모든 웹소설, 캐릭터 설정, 세계관은 전적으로 사용자에게 귀속됩니다. 서비스는 저장, 백업, 동기화 목적으로만 데이터를 보관합니다.

📌 중요: 서비스는 사용자의 명시적 승인 없이 작품을 공개, 수정, 배포하지 않습니다.

5.2 채팅 기록 및 AI 상호작용

  • 모든 AI와의 대화 기록은 사용자의 계정에만 저장됩니다
  • 채팅 기록은 서비스 개선 목적으로 분석될 수 있습니다
  • 분석 시 개인 식별 정보는 제외됩니다
  • 사용자는 언제든 채팅 기록을 전체 또는 선택적으로 삭제할 수 있습니다

5.3 데이터 백업

서비스는 다음의 목적으로 자동 백업을 실행합니다:

  • 시스템 장애 시 데이터 복구
  • 기기 분실/도난 시 다른 기기에서 복구
  • 보안 사고 예방

백업 데이터는 암호화되어 보관되며, 서비스 운영자도 열람할 수 없습니다.

5.4 데이터 내보내기

사용자는 언제든 자신의 모든 데이터를 내보낼 수 있습니다:

  • JSON 형식: 원본 데이터 구조 유지
  • PDF 형식: 인쇄 및 출판용
  • TXT 형식: 텍스트 에디터용

내보내기는 설정 → 데이터 관리 → '내 데이터 다운로드'에서 가능합니다.

5.5 계정 탈퇴 시 데이터 처리

탈퇴 직후:

  • 개인정보 (이름, 이메일, 휴대폰) 즉시 삭제
  • 결제 정보 즉시 삭제
  • 프로필 정보 즉시 삭제

유예 기간 (30일):

  • 작품 데이터는 30일간 보관
  • 이 기간 동안 복구 가능
  • 30일 후 영구 삭제
⚠️ 주의: 탈퇴 후 30일이 지나면 복구 불가능합니다. 탈퇴 전에 꼭 데이터를 내보내세요.

5.6 데이터 분석 및 개선

서비스는 다음의 목적으로 통계 정보를 수집합니다 (개인 식별 불가능):

  • AI 집필 기능 개선
  • UI/UX 최적화
  • 버그 수정
  • 새로운 기능 개발

5.7 보안 유지

모든 데이터는 다음과 같이 보호됩니다:

  • 전송 중: SSL/TLS 암호화
  • 저장 중: AES-256 암호화
  • 접근 제어: 역할 기반 권한 관리
  • 감시: 24/7 보안 모니터링

5.8 데이터 정책 변경

데이터 처리 방식이 변경될 경우 30일 전 공지됩니다.
